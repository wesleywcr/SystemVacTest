# 🏃Getting Started

```sh
#  Clone Repository
$ git clone https://github.com/wesleywcr/SystemVacTest.git
```

```sh
# Run Web aplication
cd web
# Install Dependencies
$ yarn install
# Start application
$ yarn dev

# Run API
$ cd server
$ yarn server
```
